Jingxi 检测工具箱

标准号提取 · 查询验证 · 下载管理 · 一站式解决方案

当前版本: v1.0

📋 功能介绍

📄

标准号提取

从 PDF、Word、Excel、文本、CSV 及图片文件中自动提取标准号，支持 OCR 识别扫描件，自动显示置信度

🔍

标准有效性查询

查询标准的有效性状态，包括现行、废止、即将实施等，7个平台自动切换，数据权威可靠

⬇️

标准电子版下载

从多个源查找并下载标准文件，提供在线阅读链接，支持食品伙伴网、GBT标准网等多个下载渠道

📊

客户端管理

分布式客户端管理，支持多设备同时使用，心跳续期机制保障安全，实时广播消息推送

🔐

加密配置机制

配置本地加密存储，密钥通过心跳实时续期，断网自动失效，无需复杂许可证系统

🚀 开发中的功能（即将上线）

  • 标准下载源增强 - 加强技术文件的下载和在线阅读功能
  • QC图监控系统 - 自动识别项目，制作质控图，数据异常警报
  • 检测计量招投标系统 - 招投标信息采集发布
  • CMA CNAS题库 - 评审员题库及智能问答
  • 不确定度评估 - 输入检测项目，自动生成不确定度评定方案

📖 使用方法

1

启动服务

运行 ./start.sh 启动服务端和客户端，服务端端口 14114

2

提取标准号

上传文件（PDF/Word/Excel/图片等），自动提取标准号，显示置信度和页码

3

查询有效性

输入标准号查询有效性状态，多平台自动切换，结果详细可靠

4

下载电子版

选择标准号，下载对应文件或获取在线阅读链接

💻 系统要求

操作系统 Linux (推荐Ubuntu/Debian)
Python Python 3.8+
依赖 Flask, Playwright, Selenium
网络 需要持续联网（密钥续期）
系统工具 LibreOffice, antiword, catdoc

⬇️ 下载客户端

Gitee Release（国内首选） Git仓库镜像，国内访问速度快，进入网页选择对应系统下载 GitHub Release GitHub官方源，全球访问 Linux 版下载 支持 Ubuntu、Debian、CentOS 等主流发行版 macOS 版下载 支持 macOS 10.15 及以上版本 Windows 版下载 支持 Windows 10 1809 及以上 64位系统

💡 温馨提示

  • 客户端必须持续联网，断网5分钟后自动失效
  • 标准查询需要 Selenium + ChromeDriver
  • DOC文件转换需要 LibreOffice (soffice)
